<div align="center">
  <h1>Q_lnodb</h1>
</div>

**Q_lnodb** means "Fast least number of dollar bills". It is the world's fastest
function to calculate the least number of dollar bills needed to pay a given
amount of money.

<div align="center">
  <h2>Example</h2>
  <h3>Input:</h3>
  <p>$127.25</p>
  <h3>Output:</h3>
  <p>You need one $127.25 bill.</p>
</div>
